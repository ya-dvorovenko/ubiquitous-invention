GraphQL for Frontend

Why GraphQL?

Sui's JSON-RPC API is deprecated and will be removed in future versions. GraphQL provides a more efficient, typed, and flexible way to query blockchain data. It allows us to fetch exactly the data we need in a single request, reducing network overhead and improving application performance.

query GetCreatorProfile($id: SuiAddress!) { object(address: $id) { asMoveObject { contents { json } } } }
